# Padding

```
Padding(
    padding: EdgeInsets.all(크기)) // 사방으로 패딩 적용
    padding: EdgeInsets.symmetric(horizontal:크기, verticle: 크기) // 수평, 수직으로 패딩 적용
    padding: EdgeInsets.only(left: 크기, right: 크기, top: 크기, bottom: 크기) // 커스터마이징 패딩 적용
    padding: EdgeInsets.fromLTRB(크기, 크기, 크기, 크기) // 사방으로 패딩 적용 (다른 크기로)
```
